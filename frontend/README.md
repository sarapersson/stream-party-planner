# StreamParty Planner Frontend

Minimal React 19, TypeScript, and Vite application shell for StreamParty Planner.

## Runtime

Use Node.js 24 LTS. The repository root `.nvmrc` is set to `24`.

## Local Commands

```bash
npm install
npm run dev
npm run build
```

During local development, Vite proxies `/api` requests to the backend at `http://localhost:8080`.
